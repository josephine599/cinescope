# Cinescope - Movie Discovery & Watchlist App

Cinescope is a **React-based movie discovery and watchlist application** that allows users to explore trending movies, search for films or TV shows, watch trailers, rate movies, and manage a personal watchlist. The app is designed to be simple, responsive, and user-friendly, making movie exploration easy and enjoyable.

## Features

- **Home & Trending:** View a curated list of trending movies.
- **Search:** Quickly find movies or TV shows using the search bar.
- **Movie Details:** Click on a movie to see details, including an overview, rating, and trailer.
- **Watchlist:** Add movies to a personal watchlist for later viewing.
- **Ratings:** Rate movies with a simple star-based rating system.
- **Authentication:** Simple login system to access full app features (mock authentication or Firebase).
- **Responsive Design:** Works well on desktop and mobile devices.
- **Dark/Light Theme Toggle:** Optional theming support.

## Tech Stack

- **Frontend:** React, React Router, CSS Modules
- **Backend:** FastAPI (Python)
- **Database:** SQLite (or any other supported by FastAPI)
- **API Integration:** TMDB API for fetching movie details and posters
- **Version Control:** Git & GitHub
